## ⚙️ Hướng dẫn build và chạy

## Trên Linux / macOS / WSL:

Ví dụ với file Car.cpp:
```bash
g++ Car.cpp -o Car

./Car
```

Ví dụ với file Circle.cpp:
```bash
g++ Circle.cpp -o Circle

./Circle
```
## Trên Windows (MinGW g++):
```bash
g++ Car.cpp -o Car.exe

Car.exe
```
Làm tương tự cho từng file khác (Book.cpp, Account.cpp, …).
